## Правила расчёта

- Система автоматически определяет сезон по выбранной дате (проверять каждый день, к какому сезону он относится)
- Цена = количество инструкторов × цена занятия × количество дней
- **Стоимость занятия на 4 часа** = стоимости 2 занятий по 2 часа
- **Занятие на 5 часов** = стоимость занятия на 2 часа + стоимость занятия на 3 часа
- Для "Личный гид" каждый дополнительный гость добавляет 14 000 ₽
- Групповые занятия: минимум 3 человека для закрытой группы

### Детские занятия
- Для расчета стоимости детских занятий умножаем количество часов на базовую стоимость 1 часа на 1 человека
- Для детей до 3-х лет включительно +20% стоит профессиональный детский инструктор

### Продвинутые инструкторы
Если требуется продвинутый уровень инструктора, к чеку прибавляем +20%, это категория ВИП инструктор.

Категории продвинутых инструкторов:
- Карвинг
- Фрирайд
- Фристайл
- Катание по буграм
- Катание в парке
- Занятие с детьми до 3-х лет (для курорта Роза Хутор)

## Пример расчёта

**Запрос**: 2 инструктора, по 2 ученика, по 3 часа, с 15.01.26 на 3 дня подряд

**Расчёт**:
- Сезон: Низкий (с 12.01.2026 до 05.02.2026)
- Занятие для двух человек (3 часа): 20 500 ₽
- Количество инструкторов: 2
- Количество дней: 3
- Формула: 2 × 20 500 × 3 = 123 000 ₽

**Итоговая стоимость**: 123 000 ₽
