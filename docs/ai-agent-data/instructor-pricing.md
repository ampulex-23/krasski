## Правила расчёта

- Система автоматически определяет сезон по выбранной дате
- Цена = количество инструкторов × цена занятия × количество дней
- Для многодневных программ применяется фиксированная стоимость курса
- Для "Личный гид" каждый дополнительный гость добавляет 11 000 ₽
- Рассвет на трассе: минимум 1, максимум 5 человек в группе
- Групповые занятия: минимум 4 человека для закрытой группы

## Пример расчёта

**Запрос**: 2 инструктора, по 2 ученика, по 3 часа, с 02.01.26 на 3 дня подряд

**Расчёт**:
- Сезон: Базовый (02.01-05.02.26)
- Занятие для двух человек (3 часа): 20 500 ₽
- Количество инструкторов: 2
- Количество дней: 3
- Формула: 2 × 20 500 × 3 = 123 000 ₽

**Итоговая стоимость**: 123 000 ₽
