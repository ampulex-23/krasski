# Инвентарь и снаряжение KRASSKI

## Категории снаряжения

### 1. Комплекты (лыжи/сноуборд + ботинки + палки/крепления)

**Детский комплект**
- Возраст: до 14 лет
- Включает: лыжи или сноуборд + ботинки + палки/крепления
- Уровень: подходит для всех уровней катания

**Комплект Эконом**
- Для начинающих и катающихся несколько раз в год
- Надёжное проверенное снаряжение
- Оптимальное соотношение цена-качество

**Комплект Комфорт**
- Для уверенно катающихся
- Снаряжение среднего и продвинутого уровня
- Лучшее управление и контроль

**Комплект Эксперт**
- Для опытных и экспертных райдеров
- Топовое снаряжение текущего сезона
- Максимальная производительность

## 2. Отдельное оборудование

### Лыжи/Сноуборды
- **Детские**: для детей до 14 лет
- **Эконом**: базовый уровень
- **Комфорт**: средний и продвинутый уровень
- **Эксперт**: профессиональный уровень

### Ботинки
- **Детские**: размеры для детей
- **Взрослые**: полный размерный ряд
- Профессиональная подгонка при выдаче

### Палки горнолыжные
- Регулируемые и фиксированные
- Подбор по росту

### Крепления для сноуборда
- Современные системы креплений
- Настройка под параметры райдера

## 3. Защита и одежда

### Маски
- **Детские**: защита от ветра и солнца
- **Взрослые**: различные линзы (солнечная погода, облачность, снегопад)

### Шлемы
- **Детские**: облегчённые, яркие цвета
- **Взрослые**: различные размеры, регулируемая посадка
- Обязательны для детей, рекомендованы для всех

### Куртки и штаны
- **Детские**: яркие, тёплые, водонепроницаемые
- **Взрослые**: технологичные материалы, мембрана

### Комбинезоны
- **Детские**: удобные, тёплые, полная защита
- **Взрослые**: для любителей фрирайда и экстремального катания

### Перчатки
- **Детские**: утеплённые, водонепроницаемые
- **Взрослые**: различные модели (трассовые, фрирайд, парк)

## 4. Прочее оборудование

### Рюкзаки
- Специальные горнолыжные рюкзаки
- Для фрирайда и бэккантри
- Крепления для лыж/сноуборда

### Защита (колени, локти, спина, запястья)
- Комплекты защиты для парка
- Защита спины для фрирайда
- Защита запястий для сноубордистов

### GoPro + SD карта 64GB
- Аренда экшн-камеры для съёмки катания
- Карта памяти включена
- Крепления для шлема/груди

## Правила подбора снаряжения

### По уровню катания
- **Первый раз / Начинающий**: Эконом или Комфорт
- **Уверенный**: Эконом, Комфорт или Эксперт
- **Экспертный**: только Эксперт

### По возрасту
- До 14 лет: детские комплекты и снаряжение
- 14+ лет: взрослые комплекты

### По параметрам
- **Рост**: для подбора длины лыж/сноуборда
- **Вес**: для настройки креплений
- **Размер обуви**: для подбора ботинок
- **Уровень катания**: для категории снаряжения
