# Информация о курортах Красной Поляны

## Общая информация

KRASSKI работает на всех курортах Красной Поляны:
- Роза Хутор
- Красная Поляна
- Газпром Альпика
- Газпром Лаура

## Роза Хутор

**Описание**: Один из крупнейших горнолыжных курортов России, построенный к Олимпиаде 2014 года.

**Особенности**:
- Современная инфраструктура
- Большой выбор трасс для всех уровней
- Развитая инфраструктура: рестораны, магазины, развлечения
- Отличная система подъёмников

**Цены на инструкторов**: Едины для всех курортов, определяются сезоном (Базовый/Пик)

**Детская школа KRASKIDS**:
- Индивидуальное обучение от 2,5 лет (лыжи) и от 5 лет (сноуборд)
- Групповое обучение от 5 лет
- Специальные сезонные тарифы

## Красная Поляна

**Описание**: Курорт с богатой историей, один из первых в регионе.

**Особенности**:
- Разнообразие трасс
- Живописные виды
- Доступные цены
- Уютная атмосфера

**Цены на инструкторов**: Едины для всех курортов, определяются сезоном (Базовый/Пик)

**Детская школа KRASKIDS**:
- Индивидуальное обучение с вариативными ценами по сезону (Низкий/Высокий/Пик)
- Групповое обучение 3 часа

## Газпром (Альпика и Лаура)

**Описание**: Современный горнолыжный комплекс с двумя зонами катания.

**Особенности**:
- Альпика — зона для начинающих и среднего уровня
- Лаура — более сложные трассы, фрирайд
- Качественная инфраструктура
- Хорошие условия для обучения

**Цены на инструкторов**: Едины для всех курортов, определяются сезоном (Базовый/Пик)

**Детская школа KRASKIDS**:
- Индивидуальное обучение с вариативными ценами по сезону (Низкий/Высокий/Пик)
- Групповое обучение 3 часа

## Важная информация

**Гарантия честной цены**: KRASSKI работает по ценам курортов

**Трансфер**: Доступен на все курорты
- Бесплатно при заказе от 5000₽ (прокат + инструктор)
- Выбор класса авто: комфорт, бизнес, минивэн, премиум

**Доставка снаряжения**: Бесплатно на все курорты при заказе от 5000₽

**Скипасс**: Приобретается отдельно в кассах курортов (не включён в стоимость обучения)
