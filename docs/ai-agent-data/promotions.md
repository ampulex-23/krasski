# Акции и спецпредложения KRASSKI

## Комплексный пакет «Всё включено»

Полный пакет для комфортного катания на горных лыжах или сноуборде.

### Что включено

- **Полный комплект снаряжения**: одежда, маска, шлем, лыжи+палки+ботинки / борд+крепления+ботинки
- **Занятие с инструктором**: 1,5 часа индивидуально или 2 часа в группе (при наличии групп)
- **Скипасс**: включен на курорт Красная Поляна
- **Трансфер**: 
  - С отеля до проката
  - С проката до курорта
  - С курорта до проката

### Стоимость пакета (1 человек)

| Сезон | Стоимость |
|---|---|
| **Низкий сезон** | **19 900 ₽** |
| **Высокий сезон** | **22 900 ₽** |

**Сезонность**:
- **Низкий сезон**: до 25.12.2025, с 12.01.2026 до 05.02.2026, с 10.03.2026 и до конца сезона
- **Высокий сезон**: с 26.12.2025 до 11.01.2026, с 06.02.2026 до 09.03.2026

### Как заказать

Система автоматически определяет сезон по выбранной дате. Пользователь указывает:
- Дату
- Количество гостей (от 1)
- Данные для трансфера (отель, контакт, время забора)
- Параметры для подбора снаряжения (рост, вес, размер обуви)
