## Ключевые данные для AI-агента

### Сезонность

**Прокат снаряжения:**
- Высокий сезон: новогодние праздники, весь февраль
- Низкий сезон: остальные даты

**Инструкторы (все курорты):**
- Сезон Базовый: 02.01-05.02.26, 10.03-26.06, 10.09-конец сезона
- Сезон ПИК: 26.12.25-11.01.26, 06.02.26-09.03.26

**KRASKIDS (Роза Хутор):**
- Базовый: до 27.12, 13.01–07.02, с 11.03
- Пик: 28.12–12.01, 08.02–10.03

**KRASKIDS (Красная Поляна/Газпром):**
- Низкий: до 27.12, с 08.04
- Высокий: 13.01–31.01, 13.03–07.04
- Пик: 27.12–12.01, 01.02–12.03

### Важные правила для AI

1. **Скидки на инструкторов НЕ предоставляются** — только на прокат при бронировании инструктора (10%)

2. **Максимальная скидка на прокат: 20%**

3. **Бесплатная доставка** снаряжения и трансфер от 5000₽

4. **Политика возврата:**
   - 72+ часов: 100% возврат
   - 24-72 часа: 50% возврат
   - <24 часов: возврат невозможен

5. **Скипасс НЕ включён** в стоимость обучения, покупается отдельно

6. **Цены на инструкторов едины** для всех курортов (Роза Хутор, Красная Поляна, Газпром)

7. **Детям до 14 лет** — детские тарифы на прокат

8. **AI-бот переключает на оператора** по запросу клиента

## Использование данных

AI-агент должен использовать эти файлы для:

1. **Ответов на вопросы клиентов** о ценах, услугах, курортах
2. **Расчёта стоимости** проката и занятий с учётом сезона и скидок
3. **Помощи в бронировании** с объяснением процесса
4. **Консультаций** по выбору снаряжения и инструкторов
5. **Информирования** об акциях и дополнительных услугах
6. **Разъяснения политики** возврата и изменения бронирования
